# easyRaschBayes: Bayesian Rasch Analysis Using 'brms'

Provides functions to reproduce classic Rasch analysis features using
Bayesian item response theory (IRT) models fitted with 'brms'. Supports
both dichotomous Rasch models and polytomous partial credit models.
Features include posterior predictive item fit, conditional infit,
person fit, differential item functioning (DIF), local dependence
assessment via Q3 residual correlations, dimensionality assessment via
residual principal components analysis, person-item targeting plots,
item category probability curves, and reliability via Relative
Measurement Uncertainty.
